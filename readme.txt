A simple phonebook plugin to manage and organize your contacts within WordPress.

== Description ==

**Phonebook** is a lightweight and easy-to-use WordPress plugin that allows you to create and manage a list of contacts directly from your WordPress dashboard. Whether youâ€™re maintaining a personal address book or a small CRM system, Phonebook provides a simple UI to manage contact details including name, phone number, email, and notes.

**Key Features:**

- Add, edit, and delete contacts
- Store contact information including name, phone, email, and custom notes
- Search and filter contacts easily
- Admin interface using native WordPress UI
- Fully translatable and ready for localization

Perfect for site administrators, small businesses, or personal users looking for a minimal contact management tool.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/phonebook` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to **Phonebook > Contacts** to start managing your contacts.

== Frequently Asked Questions ==

= Is Phonebook compatible with the latest version of WordPress? =  
Yes, Phonebook is tested up to the latest stable release of WordPress.

= Can I export my contact list? =  
Currently, exporting is not available. This feature may be added in a future release.

= Can I customize the fields? =  
The current version supports basic fields. Future versions may allow custom fields through filters or settings.
